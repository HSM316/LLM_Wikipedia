'''ilastik'''<ref>{{cite book|last=Sommer|first=C|author2=Straehle C |author3=Koethe U |author4=Hamprecht FA  |title=ilastik: Interactive Learning and Segmentation Toolkit|journal=IEEE International Symposium on Biomedical Imaging|year=2011|pages=230–33|doi=10.1109/ISBI.2011.5872394|isbn=978-1-4244-4127-3|s2cid=206949135}}</ref>  is a user-friendly free [[Open-source software|open source]] software for [[image classification]] and [[Segmentation (image processing)|segmentation]]. No previous experience in image processing is required to run the software.

==Features==
ilastik allows user to annotate an arbitrary number of classes in images with a [[Mouse (computing)|mouse]] interface. Using these user annotations and the generic ([[nonlinear]]) image [[Feature (computer vision)|features]], the user can train a [[random forest]] classifier. ilastik has a [[CellProfiler]] module to use ilastik classifiers to process images within a [[CellProfiler]] framework.

==History==
ilastik was first released in 2011 by scientists at the Heidelberg Collaboratory for Image Processing (HCI), [[University of Heidelberg]].

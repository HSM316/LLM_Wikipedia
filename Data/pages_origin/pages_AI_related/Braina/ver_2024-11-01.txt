'''Braina''' is a [[virtual assistant]]<ref name="raconteur">{{cite web | last=King | first=Leo | title=Top 8 virtual personal assistants| website=Raconteur | date=15 December 2015 | url=https://www.raconteur.net/report/artificial-intelligence-for-business/top-8-virtual-personal-assistants/ | archive-url=https://web.archive.org/web/20230726225438/https://www.raconteur.net/report/artificial-intelligence-for-business/top-8-virtual-personal-assistants/ | archive-date=26 July 2023 | url-status=dead}}</ref><ref name="mostar">{{cite journal|author1=Igor Bošnjak |author2=Luka Šaravanja |author3=Eva Čuljak |author4=Željko Stojkić |title = Planning and implementation of Digital Assistance System at University of Mostar Learning Factory | issue = 2021 | pages= 3–4 | publisher=[[SSRN]] | year=2021 | doi=10.2139/ssrn.3858378 | journal = 11th Conference on Learning Factories, CLF2021 |s2cid=242604709 }}</ref> and speech-to-text dictation<ref name="zdnet"/> application for [[Microsoft Windows]] developed by Brainasoft.<ref name="TechRadar">{{cite web | author=Mark Pickavance | title=Braina Pro review | date=21 April 2022 | publisher=[[Future plc]] | accessdate=2023-08-09 | url=https://www.techradar.com/reviews/braina-pro}}</ref> Braina uses [[natural language interface]],<ref name="procedia">{{cite journal|author1=Vladimir A. Fomichov |author2=Alexander A. Razorenov |title = The Design of A Natural Language Interface for File System Operations on the basis of a Structured Meanings Model |journal = Procedia Computer Science |volume = 31|issue = 2014 |pages= 1005–1011 |doi=10.1016/j.procs.2014.05.353| publisher=[[Elsevier]] |year=2014 | doi-access=free}}</ref> [[speech synthesis]], and [[speech recognition]] technology<ref name="speech recognition">{{cite web| title=Braina Speech Recognition Software| publisher=Braina| access-date=2023-09-20| url=https://www.brainasoft.com/braina/speech-to-text.html}}</ref> to interact with its users and allows them to use natural language sentences to perform various tasks on a computer. The name Braina is a short form of "Brain Artificial".<ref name="current_era_chatbots">{{cite book|author1=Robert Ciesla |title = The Current Era of Chatbots, From ELIZA to ChatGPT | doi=10.1007/978-3-031-51004-5_4| publisher=[[Springer Publishing]] | year=2024 | date=14 January 2024}}</ref><ref name="braina">{{cite web |title=Braina Homepage |accessdate=2015-01-10 |url=https://www.brainasoft.com/braina/}}</ref>

Braina is marketed as a [[Microsoft Copilot]] alternative.<ref name="copilot_alternative">{{cite web |title=Microsoft Copilot Alternative for PC|accessdate=2024-01-28 |url=https://www.brainasoft.com/braina/copilot-vs-braina.html}}</ref> It provides a voice interface for several locally run<ref name="localllm">{{cite web |title=Run LLM locally on PC|accessdate=2024-07-31 |url=https://www.brainasoft.com/braina/run-ai-language-models-locally-on-cpu-or-gpu.html}}</ref> and cloud [[large language model]]s, including [[GPT-4o]], [[Gemini (language model)|Gemini 1.5 Pro]], Anthropic's [[Claude (language model)|Claude]] Sonnet and Opus, Meta's [[Llama 3]], and [[Mistral AI|Mistral]], while attempting to improve data privacy.<ref name="current_era_chatbots"/> Braina also allows responses from its in-house large language models like Braina Swift and Braina Pinnacle.<ref name="chatgptforpc">{{cite web |title=ChatGPT for PC |accessdate=2023-08-09 |url=https://www.brainasoft.com/braina/llm-ai-chat-for-pc.html}}</ref> It has an "Artificial Brain"<ref name="current_era_chatbots"/> feature that provides persistent memory support for supported LLMs.<ref name="artificialbrain">{{cite web |title=Persistent Memory for LLM – Personal AI|accessdate=2024-01-28 |url=https://www.brainasoft.com/braina/artificial-brain.html}}</ref>

== Features ==
Braina provides is able to carry out various tasks on a computer, including automation.<ref name="indiaai.gov.in" /><ref name="hubspot">{{cite web | author=Clifford Chi | title=8 Voice-to-Text Software That'll Help You Work Faster | accessdate=2024-02-01 | url=https://blog.hubspot.com/marketing/voice-to-text}}</ref> Braina can take commands inputted through typing or through dictation<ref name="zdnet"/><ref name="dictation software">{{cite web |title=Dictation software for PC |accessdate=2024-01-28 |url=https://www.brainasoft.com/braina/dictation-software.html}}</ref><ref name="indiaai.gov.in"/><ref name="LifeWire">{{cite web | author=Stacey Nguyen | title=8 Voice-to-Text Software That'll Help You Work Faster | publisher=[[Lifewire]] | accessdate=2024-02-01 | url=https://www.lifewire.com/best-voice-to-text-apps-4583053}}</ref> to store reminders, find information online, perform mathematical operations, open files, [[Text-to-image model|generate images from text]], transcribe speech, and control open windows or programs.<ref name="MakeUseOf">{{cite web |author=Joel Lee |title=Windows 10 Transformation Pack Gives a Facelift to Windows 7 & 8 |date=25 June 2015 |publisher=MakeUseOf |accessdate=2015-06-28 |url=https://www.makeuseof.com/tag/windows-10-transformation-pack-gives-facelift-windows-7-8/}}</ref><ref name="Brainamusic">{{cite web |title=Braina Music – Search and Listen to Song |publisher=Braina |access-date=2018-03-29 |url=https://braina.me/music/}}</ref><ref name="TechRadar" /><ref name="makeuseof2">{{Cite web |author=Joel Lee |title=Windows 10 Transformation Pack Gives a Facelift to Windows 7 & 8 |date=25 June 2015 |publisher=MakeUseOf |accessdate=2015-06-28 |url=https://www.makeuseof.com/tag/windows-10-transformation-pack-gives-facelift-windows-7-8/}}</ref> Braina adapts to user behavior over time with a goal of better anticipating needs.<ref name="indiaai.gov.in">{{cite web |title=Analysing Braina - An intelligent personal assistant |accessdate=2024-01-29 |url=https://indiaai.gov.in/article/analysing-braina-an-intelligent-personal-assistant}}</ref>

=== Speech-to-text dictation ===
Braina Pro can type spoken words into an active window at the location of a user's cursor.<ref name="dictation software"/><ref name="indiaai.gov.in"/><ref name="LifeWire"/> Its [[speech recognition]] technology supports more than 100 languages and dialects<ref name="mostar"/><ref name="current_era_chatbots"/><ref name="thewindowsclub">{{cite web |title=Best Free Artificial Intelligence software for Windows 11/10|date=18 April 2022 |accessdate=2024-01-29 |url=https://www.thewindowsclub.com/artificial-intelligence-software-for-windows-10}}</ref><ref name="indiaai.gov.in" /> and is able to isolate the recognition of a user's voice from disturbing environmental factors such as background noise,<ref name="asr_aviation">{{cite journal|author1=Gaojian Huang |author2=Brandon J. Pitts, Ph.D. |title = Automated Speech Recognition Technology to Support in Flight Weather Related Communication for GA Pilots. | issue = 2019 |pages= 469 | publisher=[[Wright State University]] | year=2019 | journal= 20th International Symposium on Aviation Psychology, 468–473 | url=https://corescholar.libraries.wright.edu/cgi/viewcontent.cgi?article=1078&context=isap_2019 }}</ref> other human voices, or external devices. Braina can also be taught to dictate uncommon legal, medical, and scientific terms.<ref name="indiaai.gov.in" /><ref name="hearing_journal">{{cite journal|author1=Dr. Kader Sara Esther |author2=Eckert, Anne M. | author3=Dr. Gural-Toth |title = Voice-to-Text Technology for Patients with Hearing Loss | issue = 2 |volume= 74 | publisher=[[Lippincott Williams & Wilkins]] | year=2021 | journal= The Hearing Journal |pages=11,14,15 | doi=10.1097/01.HJ.0000734212.09840.d7 }}</ref> Users can also teach Braina uncommon names and vocabulary.<ref name="LifeWire"/> Users can edit or correct dictated text without using a keyboard or mouse by giving built-in voice commands.<ref name="indiaai.gov.in" />

=== Text-to-speech ===
Braina can read aloud selected texts, such as e-books.<ref name="TechRadar" /><ref name="indiaai.gov.in"/>

=== Custom commands and automation === 
Braina can automate computer tasks.<ref name="hubspot"/> It lets users create custom voice commands to perform tasks such as opening files, programs, websites, or emails, as well as executing keyboard or mouse macros.<ref name="TechRadar" /><ref name="makeuseof">{{cite web | author=Neeraj Paruthi | title=Why Braina Is My Preferred Virtual Assistant on Windows | date=15 September 2021 | accessdate=2024-02-01 | url=https://www.makeuseof.com/braina-windows-virtual-assistant/}}</ref><ref name="digitaltrends">{{cite web | author=Daniel Martin | title=Best dictation software | date=24 March 2021 | accessdate=2024-02-01 | url=https://www.digitaltrends.com/computing/best-dictation-software/#no3 |publisher=Digital Trends}}</ref><ref name="indiaai.gov.in"/><ref name="advancedaiautomation">{{cite web | title=Self operating computer with advanced AI automation | accessdate=2024-06-12 | url=https://www.brainasoft.com/braina/advanced-ai-automation.html}}</ref>

=== Transcription ===
Braina can transcribe media file formats such as [[WAV]], [[MP3]], and [[MP4]] into text.<ref name="braina_transcription">{{cite web |title=Automatic Video & Audio Transcription |access-date=2024-02-01 |url=https://www.brainasoft.com/braina/transcribe-audio-video-files-to-text.html}}</ref>

=== Notes and reminders ===
Braina can store and recall notes and reminders. These can include scheduled or unscheduled commands, checklist items, alarms, chat conversations, memos, website snippets, bookmarks, contacts.<ref name="indiaai.gov.in"/><ref name="TechRadar" /><ref name="braina_notes">{{cite web | title=Note taking with voice commands | accessdate=2024-02-01 | url=https://www.brainasoft.com/braina/notes.html}}</ref>

=== Image generation ===
Brainasoft states that Braina can generate images from text using [[text-to-image model]]s including [[Stable Diffusion]] and [[DALL-E]].<ref name="Braina"/>

== Distributions ==
Braina is distributed in multiple modes. These include Braina Lite, a freeware version with limitations,<ref name="zdnet">{{cite web |title=Free Artificial Intelligence (AI) software for your PC |website=[[ZDNet]] |accessdate=2024-01-29 |url=https://www.zdnet.com/article/free-artificial-intelligence-ai-software-for-your-pc/}}</ref> and premium versions Braina Pro,<ref name="indiaai.gov.in" /> Pro Plus, and Pro Ultra.<ref name="braina_download">{{cite web |title=Download Braina Virtual Assistant|accessdate=2024-01-31 |url=https://www.brainasoft.com/braina/download.html}}</ref>

'''revoscalepy''' is a [[machine learning]] package in [[Python (programming language)|Python]] created by [[Microsoft]]. It is available as part of Machine Learning Services in [[Microsoft SQL Server]] 2017 and Machine Learning Server 9.2.0 and later.<ref name="revoscalepy-intro"/>

The package contains functions for creating [[linear model]], [[logistic regression]], [[random forest]], [[decision tree]] and [[Gradient boosting|boosted decision tree]], in addition to some summary functions for inspecting data.<ref name="revoscalepy-doc"/>   Other machine learning algorithms such as neural network are provided in microsoftml, a separate package that is the Python version of MicrosoftML.<ref name="microsoftml-doc"/>

revoscalepy also contains functions designed to run machine learning [[algorithm]]s in different compute contexts, including SQL Server, [[Apache Spark]], and [[Hadoop]].<ref name="revoscalepy-doc"/>

'''JSON Feed''' is a [[Web feed]] file format for [[Web syndication]] in [[JSON]] instead of [[XML]] as used by [[RSS]] and [[Atom (Web standard)|Atom]].<ref>{{cite web |url=https://jsonfeed.org/version/1 |title=JSON Feed: Version 1 |date=2017-05-17 |author1=Brent Simmons |author2=Manton Reece}}</ref>

A range of software libraries and web frameworks support content syndication via JSON Feed.<ref>{{cite web |url=https://jsonfeed.org/code |title=JSON Feed: Code |author1=Brent Simmons |author2=Manton Reece}}</ref> Supporting clients include [[NetNewsWire]], [[NewsBlur]],<ref name="blog_News">{{Cite web |title=NewsBlur now supports the new JSON Feed spec |author=Samuel Clay |work=The NewsBlur Blog |date=23 May 2017 |access-date=17 July 2020 |url= https://blog.newsblur.com/post/160982162270/newsblur-now-supports-the-new-json-feed-spec}}</ref> and Reeder.<!--ref: https://moshbox.jp/?p=30715-->

Notable publishers include [[NPR]].<ref name="npr._NPRn">{{Cite web |title=NPR now supports JSON Feed! |last=Kasbergen |first=Nara |work=Medium |date= |access-date=17 July 2020 |url= https://npr.codes/npr-now-supports-json-feed-1c8af29d0ce7}}</ref>

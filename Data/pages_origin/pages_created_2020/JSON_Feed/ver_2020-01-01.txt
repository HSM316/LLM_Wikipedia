'''JSON Feed''' is a file format for [[Web syndication]] in [[JSON]] instead of XML as used by [[RSS]] and [[Atom (Web standard)|Atom]].<ref>{{cite web |url=https://jsonfeed.org/version/1 |title=JSON Feed: Version 1 |date=2017-05-17 |author1=Brent Simmons |author2=Manton Reece}}</ref>

A range of software libraries and web frameworks support content syndication via JSON Feed.<ref>{{cite web |url=https://jsonfeed.org/code |title=JSON Feed: Code |author1=Brent Simmons |author2=Manton Reece}}</ref> Supporting clients include [[NetNewsWire]] and Reeder.<!--ref: https://moshbox.jp/?p=30715-->
